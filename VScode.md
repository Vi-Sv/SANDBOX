## МИКРО-ШАГ 1.6: Создание портативного контура VS Code
Изолированное действие:

   1. Зайди в свой рабочий каталог D:\Portable_Data_Suite\.
   2. Создай там новую пустую папку с именем vscode .
   3. Распакуй всё содержимое скачанного ZIP-архива VSCode-win32-x64-...zip внутрь этой созданной папки D:\Portable_Data_Suite\vscode\ [1.4, 1.5]. (В корне папки должен появиться исполняемый файл Code.exe) [1.4, 1.5].
   4. Критическая точка контроля (Активация Portable-режима): Зайди внутрь папки D:\Portable_Data_Suite\vscode\. Создай там вручную новую пустую папку с именем data (строго маленькими буквами).

Физика процесса: Наличие пустой папки data в одном корне с файлом Code.exe является для VS Code жестким системным триггером. Обнаружив её при старте, редактор мгновенно отключает работу с глобальными каталогами Windows (%USERPROFILE%/.vscode/extensions) и начинает складывать все устанавливаемые плагины, кэш и настройки внутрь этой папки. 


Папка data создана в одном корне с Code.exe. Перенос настроек и плагинов в изолированный режим выполнен успешно.
Теперь нам нужно запустить редактор и установить официальное расширение Python, чтобы VS Code научился понимать синтаксис кода и автоматически связался с окружением Anaconda3.

## МИКРО-ШАГ 1.7: Первичный запуск и установка расширения Python
Изолированное действие:

   1. Зайди в папку D:\Portable_Data_Suite\vscode\ и запусти файл Code.exe<|Вторичный синтаксический контроль|>.
   2. На левой вертикальной панели инструментов найди иконку Extensions (четыре квадрата, где один отлетает в сторону) и кликни по ней.
   3. В верхней строке поиска вбей руками: Python.
   4. Найди в результатах официальный плагин от Microsoft (рядом с названием будет стоять синяя галочка верификации) и нажми кнопку Install.
   5. Дождись, пока кнопка сменится на шестеренку (установка завершена).
